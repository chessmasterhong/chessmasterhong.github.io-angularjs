Currently, I switch between Windows 7 and Arch Linux depending on the necessity of the situation.

About This Website

This website was created with AngularJS, a web application framework for creating dynamic, single-page applications using HTML, CSS, and JavaScript. The motivation was to experiment with new web technologies and to further improve my knowledge of AngularJS. The site is hosted on GitHub Pages. Its source code is publicly available on my GitHub repository.

Designed with minimalism and functionality in mind, each site component was carefully selected, designed, and distributed throughout the site. Several web design concepts were implemented such as Responsive Web Design and Adaptive Web Design. Optimization techniques were incorporated to maximize efficiency while delivering equivalent content. The site successfully validates against the W3C HTML5 standard. It tries to conform to "best practices" by performing code linting and following style guidelines. Automated build processes were created and utilized to expedite tedious tasks, thus directing additional focus towards site development.
